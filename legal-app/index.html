⚙️ Настройки

Для аналитических функций используется API Anthropic (Claude). Ключ хранится только в этом браузере (localStorage) и отправляется напрямую в Anthropic.
Получить ключ: console.anthropic.com → API Keys. Запросы идут с заголовком прямого браузерного доступа.

💻 Работа на нескольких устройствах

Данные (реквизиты компаний, отслеживаемые темы, история анализов) хранятся только на этом устройстве — ничего не уходит в интернет. Это сделано ради конфиденциальности договоров и кейсов. Чтобы перенести данные на рабочий/домашний ноутбук:
  1. На текущем устройстве нажмите «Экспорт» — скачается файл lawai-data-….json.
  2. Перенесите файл (флешка, почта себе, облачный диск) на другое устройство.
  3. Там откройте приложение и нажмите «Импорт», выберите файл.
🔑 API‑ключ намеренно не попадает в файл экспорта — вводите его один раз в настройках на каждом устройстве (это безопаснее).
Открыть приложение с любого устройства можно по ссылке (GitHub Pages) — там работают все ИИ‑разделы; живая лента adilet — только при локальном запуске через serve.ps1.
